## 1. 幻灯片结构

```
├── 封面页 (1 slide)
├── 目录页 (1 slide)
├── ── Part 1: 操作手册 ──
│   ├── 分隔页 "操作手册" (1 slide)
│   ├── 概述 (1-2 slides)
│   ├── 配置模块 A — 字段说明 + 操作步骤 (2-4 slides per module)
│   ├── 配置模块 B — 字段说明 + 操作步骤 (2-4 slides per module)
│   ├── ...
│   ├── FAQ (1 slide)
│   └── 快速参考 (1 slide)
├── ── Part 2: 字段对照图 ──
│   ├── 分隔页 "字段对照图" (1 slide)
│   ├── 配置模块 A — 左右截图 + 编号标注 + 图例表 (1-2 slides)
│   ├── 配置模块 B — 左右截图 + 编号标注 + 图例表 (1-2 slides)
│   └── ...
└── 尾页 (1 slide, optional)
```

## 2. Markdown → PPT 内容映射

| Markdown 内容                  | PPT 幻灯片类型                                 |
| ------------------------------ | ---------------------------------------------- |
| `# 标题`                       | 封面页（大标题 + 副标题）                      |
| `## 配置模块`                  | 分节页（模块名 + 简要说明）                    |
| 字段说明表格                   | `slide.addTable()` 表格幻灯片                  |
| 操作步骤（有序列表）           | 编号列表幻灯片（`bullet: { type: 'number' }`） |
| 注意事项（无序列表）           | 要点列表幻灯片（`bullet: true`）               |
| 截图引用 `![](images/xxx.png)` | `slide.addImage()` 嵌入截图                    |
| FAQ 表格                       | 表格幻灯片                                     |

## 3. 字段对照图幻灯片构建

字段对照图幻灯片用 pptxgenjs 原生构建（不是截取 HTML 页面）：

1. **截图嵌入**：`slide.addImage({ path: "images/xxx.png", ... })` 嵌入后台和前端截图
2. **编号标记**：带编号的小圆形 shape 叠加在截图上方（见下方标记系统）
3. **图例表格**：`slide.addTable()` 生成编号-字段-位置-说明的对照表

### 布局参考（16:9, 10" × 5.625"）

```
┌─────────────────────────────────────────────┐
│  标题: "配置模块名称 — 字段对照"             │ y=0.2
│                                              │
│  ┌─ 后台配置 ──────┐  ┌─ 前端展示 ──────┐   │ y=0.8
│  │   截图 + 标记    │  │   截图 + 标记    │   │ y=1.15, h≈2.8
│  │   x=0.5, w=4.3  │  │   x=5.2, w=4.3  │   │
│  └─────────────────┘  └─────────────────┘   │
│                                              │
│  ┌─ 图例表格 ────────────────────────────┐   │ y≈4.2
│  │ 编号 │ 后台字段 │ 前端位置 │ 说明     │   │
│  └──────────────────────────────────────┘   │
└─────────────────────────────────────────────┘
```

## 4. 标记（Marker）系统

标记用于在截图上标注字段位置，左右两侧相同编号表示对应关系。

### 颜色规则

- 红色（`E74C3C`）= 后台字段标记
- 蓝色（`3498DB`）= 前端展示标记

### 实现方式

每个标记由两个 pptxgenjs 元素叠加：

1. `addShape('ellipse', ...)` — 彩色圆形背景（带阴影）
2. `addText(num, ...)` — 白色编号文字

位置计算：将 HTML 中的百分比坐标转换为英寸坐标（基于截图在幻灯片上的实际位置和尺寸）。

```
英寸X = 截图起点X + (百分比X × 截图宽度)
英寸Y = 截图起点Y + (百分比Y × 截图高度)
```

## 5. 样式约定

### 推荐色板

```javascript
const C = {
  dark: "2C3E50", // 标题、表头背景
  accent: "E74C3C", // 强调色、后台标记
  blue: "3498DB", // 前端标记
  white: "FFFFFF", // 白色背景/文字
  light: "F8F9FA", // 浅灰背景
  textPrimary: "2C3E50",
  textSecondary: "7F8C8D",
  tableHeaderBg: "2C3E50",
  tableRow1: "FFFFFF", // 奇数行
  tableRow2: "F2F4F7", // 偶数行（斑马纹）
};
```

### 表格样式

- 表头：深色背景 + 白色粗体文字
- 数据行：斑马纹交替（白/浅灰），提升可读性
- 边框：细线（`pt: 0.5, color: 'DEE2E6'`）

### 幻灯片页脚

每页底部加一条强调色横条（`accent` 色，高度 0.175"），统一视觉节奏。

### 页码

右下角小字页码（fontSize: 8, textSecondary 色）。
